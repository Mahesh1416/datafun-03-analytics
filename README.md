## Commands to Manage Virtual Environment

For Windows PowerShell (change if using Mac/Linux)

```powershell
py -m venv .venv
.\.venv\Scripts\activate
py -m pip install --upgrade pip setuptools wheel
py -m pip install --upgrade -r requirements.txt
```
